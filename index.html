About safejaya

The proposed system is the digital-based SafeJaya application, a neighborhood security information system integrated with the Smart City concept. This application is designed to facilitate real-time reporting and handling of security incidents in the Mustika Jaya District. This system integrates GPS technology for location tracking, push notifications for emergency alerts, and cloud computing for centralized data storage. With a user-friendly interface, this application allows residents to easily report security incidents via their smartphones, while also providing monitoring access to authorities for a faster and more coordinated response.
